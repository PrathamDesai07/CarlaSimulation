    - Description: Shell script to wrap `ffmpeg` calls and produce a final MP4 from captured frames.
    - Inputs: Frame directory path, frame rate, output path.
    - Outputs: `outputs/videos/longest6_<model_id>.mp4`.
  - `outputs/videos/…`
    - Description: Folder where all generated videos per run go.

  - Selected `model.pth`.
  - Frame output directory and fps.
  - CARLA server endpoint.
